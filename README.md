# Project Structure

- **deployment**
  - Weights of models
  - Prototype.py

- **ML & DL**
  - **CNN**
    - Weights and the results in CSV files
  - **HW-NAS**
    - Search space configurations and latency estimation logs
  - **Machine_Learning_Models**
    - **Models_Results**
      - Results for 15, 6, and binary classes
    - Best_hyperparameter.md
    - Constraints_hardware_(ML).py

  - Features_extractions.md

  - Features_extractions.py

- DNN_Model.py
- Grid_search.md
- Model_ML.py

- **master**: Contains the complete project source code with the original file and folder structure for development and version tracking.
